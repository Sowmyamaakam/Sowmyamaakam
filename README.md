## Featured Projects

### 🧠 [PSYVANA — Mental Wellness Platform](https://github.com/Sowmyamaakam/PSYVANA)
> Android app + Web platform for evidence-based mental health assessment

- Clinically validated tools: **PHQ-9** (depression), **GAD-7** (anxiety), **PSS-10** (stress)
- AI-powered personalised activity recommendations based on assessment scores
- Doctor connect via **chat, audio & video** consultation
- Progress tracking, history dashboard, and crisis helpline integration
- **Stack:** Java · Android SDK · Firebase · React · Vite · TailwindCSS · Framer Motion

---

### 💸 [UPI FinAnalyzer — AI Financial Dashboard](https://github.com/Sowmyamaakam/UPI_FinAnalyzer)
> Turn UPI screenshots into actionable financial intelligence

- **OCR engine** extracts transaction data from UPI receipt screenshots — zero manual entry
- Real-time spending dashboards, category breakdowns, and smart alerts
- Group expense splitting and **AI-powered savings suggestions**
- **Stack:** Python · OCR · Firebase · HTML/CSS/JS

---

### 🎙️ [AWS Audio Transcription](https://github.com/Sowmyamaakam/AWS-Audio-Transcription-Project)
> Serverless audio-to-text pipeline with speaker diarization

- Upload audio files or record live in-browser with real-time **waveform visualizer**
- Powered by **AWS Lambda + S3 + Amazon Transcribe** — fully serverless
- **Speaker diarization** distinguishes up to 10 speakers per recording
- REST API (3 endpoints) with polling-based status tracking
- **Stack:** AWS Lambda · API Gateway · S3 · Amazon Transcribe · Python

---

### 📊 [SentimentIQ — Multi-Engine Sentiment Analysis](https://github.com/Sowmyamaakam/SentimentIQ)
> Compare 5 AI sentiment engines side-by-side in real time

- Runs **5 NLP models in parallel** — VADER, TextBlob, DistilBERT, RoBERTa, AFINN
- Emotion breakdown, confidence scoring, key phrase extraction, word-level analysis
- Industry-specific insights for e-commerce, healthcare, HR, and customer service
- Beautiful **glassmorphism dark UI** with animated bar charts
- **Stack:** React · Vite · TailwindCSS · Framer Motion · Recharts

---

### 👁️ [Glaucoma Detection — Medical AI](https://github.com/Sowmyamaakam/Glaucoma_Detection)
> Deep learning model for early glaucoma screening from retinal images

- CNN-based classification model for retinal fundus image analysis
- Assists in **early detection** of glaucoma — a leading cause of blindness
- Trained and evaluated on labelled ophthalmology datasets
- **Stack:** Python · TensorFlow/Keras · OpenCV · NumPy

---

### 🎤 [Speech-Based Depression Detection](https://github.com/Sowmyamaakam/Speech-Based-Depression-Detection)
> Detecting depression markers from voice signals using ML

- Extracts acoustic features (MFCCs, pitch, energy) from speech recordings
- Machine learning classifier trained on clinical voice datasets
- Enables passive, non-invasive mental health screening
- **Stack:** Python · librosa · scikit-learn · NumPy
